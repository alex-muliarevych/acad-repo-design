# acad-frontend

### Installation

First - install `yarn` globally:
```sh
npm install -g yarn
```

Then install project dependencies:
```sh
yarn
```
or
```sh
yarn install
```

### Development

```sh
npm start:dev
```

Then head over to `localhost:4200`!


### Production build

```sh
yarn build:production
```

Then get `.tar` bundle from `/builds` folder
